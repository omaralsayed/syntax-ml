How does Artificial Intelligence and code interact?

Machine Learning is made possible by increasing access to valuable data and compute. There has beenagrowinginterestin the field to explore how AI may interface with source code to either refine existing code or augment the capabilities of a programmer. This site is a place to play with small-scale demos of programs inspired by this field of research.

word2code

word2code uses the word2vec pre-trained weights to classify labeled code snippets in response to a natural language query. Using the pre-trained word2vec model simplified the implementation of the NLP classification.


Language Recognition

Using statistical classification, it is possible to determine the programming language of a given segment of source code. This is done through analyzing the frequency of different syntactical structures. We classified the source code of cpython, jquery, and gcc's c++ library to find the frequency of certain keywords.
